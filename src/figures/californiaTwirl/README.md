# California twirl

Parameters:
- default 4 beats

Displays like: "[California twirl]"

Final keyframe: assert each lark has a robin on his right, and that he's on her left. In the final keyframe, those two should have swapped places, and each turned 180 degrees, and they should be holding inside hands (lark's right hand, robin's left), and no other hands.

Keyframe generation: they pair takes inside hands, and drops hands with anybody else; then they swap places by going in a clockwise ellipse (exactly like box the gnat), lark turning 180 cw, robin turning 180 ccw.

Example dance: start in improper, 0 progression; California twirl twice.
