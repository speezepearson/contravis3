# Turn alone

Parameters:
- default 1 beat


Displays like: "[turn alone]"

Final keyframe: same as initial but every dancer facing the opposite direction.

Keyframe generation: larks turn 180 clockwise, robins turn 180 ccw.

Example dance: start in improper, 0 progression; step forward 0.2m over 3 beats, then turn alone, then step forward 0.2m over 3 beats, then turn alone.
