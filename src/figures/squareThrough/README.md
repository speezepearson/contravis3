# Square through two

(This is a "square through two"; a "square through four" is just two of this in a row.)

Parameters:
- default 8 beats

Displays like: "[square through]"

Equivalent to "face across; take right hands with the person in front of you; balance them; pull by right; turn (right for larks, left for robins); pull by left with the person in front of you."
