# Contravis3

A browser-based visualizer and editor for contra dance choreography, built with TypeScript, React, and Vite.

## Commands

- `npm run dev` — Start dev server
- `npm run build` — Type-check and build
- `npm run test` — Run tests (vitest)
- `npm run lint` — Lint

## Automated demos

When creating a pull request that changes the user interface (any file matching `src/App.tsx`, `src/CommandPane.tsx`, `src/SearchableDropdown.tsx`, `src/SearchableDropdown.css`, `src/renderer.ts`, `src/index.css`, or `src/main.tsx`), **always** run the `.claude/skills/demo.md` skill to generate a demo document in `demos/` before opening the PR.
